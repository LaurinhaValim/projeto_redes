- Descrição

Este projeto consiste no desenvolvimento de uma aplicação cliente-servidor utilizando sockets TCP em Python. O sistema permite a comunicação entre clientes e servidor por meio do protocolo TCP, permitindo a troca de mensagens e dados entre os dispositivos conectados.

O objetivo do trabalho é aplicar os conceitos estudados na disciplina de Redes de Computadores, como comunicação em rede, arquitetura cliente-servidor, conexões TCP, envio e recebimento de dados e tratamento de múltiplas requisições.

- Tecnologias Utilizadas

* Python
* Sockets TCP
* Programação Cliente-Servidor
